# Blue Sky CRM ????

A specialized B2B Sales & Distribution Management tool designed for high-purity chemical networks. 

## ?? Overview
This project demonstrates a modular approach to managing complex B2B sales cycles. It calculates revenue based on product purity metrics and includes automated risk management alerts for quality control.

## ??? Tech Stack
- **Node.js**: Runtime environment.
- **JavaScript (ES6+)**: Core business logic and modular imports.
- **Git/GitHub**: Version control and deployment.

## ?? Key Features
- **Purity-Adjusted Revenue Logic**: A custom algorithm that calculates batch value based on quality percentages.
- **Automated Quality Audits**: Real-time risk alerts if product purity falls below the 96% threshold.
- **Modular Architecture**: Clean separation between business math (`business.js`) and reporting (`index.js`).
